📝 Admission Test

At NPconscious, every student goes through a short admission test. This helps us understand the student's current level and personalize the learning experience.

Classes 1–5

• Live test with a teacher (30 minutes)

• Parent/guardian must be present

Classes 6–10

• Live test with a teacher (30 minutes)

• Parents can monitor(optional)

Classes 11–12

• Live test with a teacher (30 minutes)

• Parents can monitor(optional)

Engineering Mathematics (UG)

• Written test (1 hour)

Money & Mathematics

• MSQ based test

🎯 This test helps identify strengths and areas of improvement so we can guide the student effectively from the start.

To begin the admission process, please share:

• Student Name
• Grade / Class
